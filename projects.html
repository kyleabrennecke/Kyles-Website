A.R.M.S

ARMS (Andon Registration and Management System) is a web application that allows users to register and manage their andons. An andon is a visual control unit that indicates the status of a machine or process. ARMS makes it possible to update these units in groups and execute commands on them without having to remote into them individually. The application is built using PHP and MySQL and reduced andon update times from days to minutes.
